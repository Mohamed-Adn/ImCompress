# **App Name**: ImCompress

## Core Features:

- Image Upload: Allow users to upload images from their device.
- Image Preview: Display a preview of the uploaded image before and after resizing/compression.
- Resizing: Allow users to specify the desired dimensions (width and height) for resizing the image.
- Compression: Allow users to adjust the compression level (quality) of the image to reduce file size.
- AI Parameter Advisor: Employ a tool that uses generative AI to suggest optimal resizing and compression settings based on the image's content and the user's desired output file size or quality.
- Format Conversion: Offer options to convert the image to different file formats (e.g., JPEG, PNG, WebP).
- Download: Enable users to download the processed image.

## Style Guidelines:

- Primary color: Saturated purple (#A020F0) for creativity and transformation.
- Background color: Light grayish-purple (#F0E6FF) to complement the primary and offer a clean backdrop.
- Accent color: Bright green (#00FF00) to provide contrast for CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a clean and modern user interface.
- Use simple, clear icons to represent image settings and actions (e.g., resize, compress, download).
- Maintain a clean and intuitive layout, with image preview prominently displayed.
- Provide subtle animations to confirm actions (e.g., successful compression, image upload).
